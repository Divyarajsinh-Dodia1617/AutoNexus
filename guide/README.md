| I want to... | Use |
|--------------|-----|
| Improve a metric autonomously | `/autonexus` |
| Don't know what metric to use | `/autonexus:plan` |
| Get expert opinions before starting | `/autonexus:predict` |
| Predict then debug | `/autonexus:predict --chain debug` |
